# Notes Application

## Overview

Notes Application is a web-based note-taking application designed to help users create, organize, and manage their personal notes efficiently. The application provides a simple and user-friendly interface with support for rich text formatting, folders, Markdown syntax, searching, sorting, and automatic data persistence.

The project was developed as part of the Web Programming course to demonstrate front-end development concepts, data management, and browser storage mechanisms.

---

## Features

### Note Management

* Create new notes with a custom title and text content.
* Edit and delete existing notes.
* Support rich text formatting including:

  * Different text colors.
  * **Bold** text.
  * *Italic* text.
  * Underlined text.
* Automatically save all modifications.

---

### Folder Organization

* Create folders to organize notes.
* Rename and delete folders.
* Move notes between folders.
* Support a single-level folder structure (nested folders are not allowed).

---

### Sidebar Navigation

* A permanently visible sidebar for quick access.
* Display all folders and notes.
* Navigate between notes and folders easily.
* Create new notes and folders directly from the sidebar.
* Display pinned notes at the top of the list.

---

### Markdown Support

The application supports simplified Markdown syntax, including:

* Headings (`#`, `##`, etc.)
* Ordered and unordered lists.
* Bold text (`**text**` or `__text__`).
* Italic text (`*text*` or `_text_`).
* Links (`[text](url)`).
* Inline code (`code`).
* Code blocks.

Advanced Markdown features such as syntax highlighting and mathematical expressions are not included.

---

### Note Browsing and Editing

* View all created notes.
* Open notes directly from the sidebar.
* Edit note content in the main editor area.
* Automatically save changes while typing.

---

### JSON Import & Export

* Export all application data as a JSON file.
* Import JSON files to restore notes and folders.
* Transfer notes between different devices.

---

### Pinning Favorite Notes

* Pin an unlimited number of favorite notes.
* Display pinned notes at the top of the sidebar.
* Identify pinned notes using a visual indicator (such as a pin icon).
* Remove notes from the pinned section at any time.

---

### Search and Sorting

* Search notes by title or content.
* Filter notes dynamically while typing.
* Sort notes alphabetically.
* Sort notes by creation date or last modification date.

---

### Automatic Local Storage

* Save all changes automatically using browser Local Storage.
* Preserve notes after closing or reopening the application.
* Eliminate the need for a manual save button.

---
